# rails-container

rails-container is a portable Ruby on Rails environment for developers.
This repository represents a container building process as the step by step commits.

# make commands

build Docker image:

```
% make build
```

bundle install:

```
% make install
```

rails new:

```
% make new
```

check rails version:

```
% make run
```

# change Ruby version

- modify .ruby-version
- make build
